# Python Language Overview

## What Python Is
Python is a high-level, general-purpose programming language whose design philosophy emphasizes readability and simplicity. It is dynamically typed, garbage-collected, and supports multiple paradigms including procedural, object-oriented, and functional programming.

Python is widely used for automation, web development, APIs, scripting, data science, machine learning, scientific computing, and education.

## Current Version Context (2026)
As of 2026, active support covers Python 3.10 through 3.14, and Python 3.15 is in alpha development with a stable release expected in late 2026. Modern Python follows an annual release cycle and a support lifecycle with full support followed by security-only support.

## Brief History
- Late 1980s: Guido van Rossum began Python at CWI as a successor to ABC.
- 1991: first public release (0.9.0).
- 2000: Python 2.0 added major features like list comprehensions and improved Unicode support.
- 2008: Python 3.0 introduced major language cleanup and backward-incompatible changes.
- 2020: Python 2 reached end-of-life.

Python's name comes from Monty Python's Flying Circus.

## Design Philosophy
Python favors explicit, readable code. The Zen of Python (PEP 20) summarizes the language values, including:
- Explicit is better than implicit.
- Simple is better than complex.
- Readability counts.
- There should be one obvious way to do it.
