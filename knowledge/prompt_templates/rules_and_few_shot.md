# Tiny Model Rules and Few-Shot Tips

## Important Rules For 2M Models
- Use short sentences.
- Give clear examples.
- Limit output length.
- Keep tasks very specific.
- Do not ask broad "why" questions.
- Do not expect long explanations.
- Do not use long paragraphs.

## Pro Tip: Few-Shot Is Better
Bad:
Classify this text.

Good:
Input: hello -> greet
Input: buy item -> buy
Input: I need help -> help
