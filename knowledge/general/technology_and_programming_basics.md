# Technology and Programming Basics

## Internet and Web Basics
The internet is the network infrastructure. The web is a service on top of the internet using HTTP/HTTPS. A URL identifies a resource location, and a domain name maps to an IP address through DNS.

## Computer Basics
A CPU executes instructions, memory (RAM) stores active data, and storage keeps persistent data. Operating systems manage hardware, files, processes, and permissions.

## Programming Basics
A variable stores a value, a function groups reusable logic, and a loop repeats actions. Conditional branches choose behavior based on boolean conditions.

## Python Scripting Basics
A basic Python script should include an entry point with if __name__ == '__main__'. Keep scripts small, readable, and split logic into functions where possible.
